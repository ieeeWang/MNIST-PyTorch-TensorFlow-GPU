# MNIST-PyTorch-TensorFlow-GPU

The [MNIST](http://yann.lecun.com/exdb/mnist/) dataset of handwritten digits for image classification using CNNs.



You will need to install dependencies:

- TensorFlow2.x
- pytorch > 1.0
- Numpy
